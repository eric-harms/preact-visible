# preact-visible

A tiny wrapper component that determines if its children are visible.

	import Visible from "preact-visible";

    const Example = () => (
	    <Visible isVisible={true}>
  	        <h1>
  	        	Heading
  	        </h1>
        </Visible>
    );
    
## Installation

	npm install preact-visible
